# 📝 Guia de Formatação Markdown para Posts

## ✨ Sistema de Auto-Formatação

O sistema agora detecta automaticamente a formatação markdown e estrutura o conteúdo!

---

## 🎯 Formatos Suportados

### 1️⃣ **Títulos/Headings**

#### Opção A: Markdown (##, ###)
```markdown
## Título Principal
### Subtítulo
```

#### Opção B: MAIÚSCULAS
```
TÍTULO PRINCIPAL
```

**Regras:**
- Mínimo 4 caracteres
- Máximo 100 caracteres
- Apenas letras, números, espaços e pontuação básica

---

### 2️⃣ **Parágrafos**

```
Escreva normalmente.
Várias linhas serão juntadas em um único parágrafo.

Deixe uma linha em branco para criar um novo parágrafo.
```

**Resultado:**
- Linhas consecutivas = 1 parágrafo
- Linha vazia = Novo parágrafo

---

### 3️⃣ **Listas com Bullets** •

#### Opção A: Hífen (-)
```markdown
- Primeiro item
- Segundo item
- Terceiro item
```

#### Opção B: Asterisco (*)
```markdown
* Primeiro item
* Segundo item
* Terceiro item
```

#### Opção C: Bullet (•)
```markdown
• Primeiro item
• Segundo item
• Terceiro item
```

**Todas funcionam!**

---

### 4️⃣ **Listas Numeradas**

```markdown
1. Primeiro passo
2. Segundo passo
3. Terceiro passo
```

**Renderiza como:**
1. Primeiro passo
2. Segundo passo
3. Terceiro passo

---

### 5️⃣ **Citações/Quotes**

```markdown
> Esta é uma citação importante.
> Use o símbolo de maior (>) no início da linha.
```

**Renderiza:**
> Citação destacada com estilo especial

## 🎨 Como Será Renderizado

### Títulos
- Fonte grande e destacada
- Cor azul Gabardo
- Espaçamento generoso

### Parágrafos
- Texto legível (18-20px)
- Cor branca suave
- Espaçamento entre linhas

### Listas
- **Bullets:** Pontinhos brancos
- **Numeradas:** Números sequenciais
- Animação ao scroll
- Espaçamento uniforme

### Citações
- Fundo destacado
- Borda esquerda colorida
- Fonte em itálico
- Destaque visual

---

## ✅ Dicas de Uso

### ✓ Faça:
- Use títulos para separar seções
- Mantenha parágrafos concisos (3-5 linhas)
- Use listas para informações sequenciais
- Adicione citações para destacar insights
- Deixe linhas vazias entre blocos diferentes

### ✗ Evite:
- Blocos de texto muito longos sem quebras
- Muitos títulos seguidos sem conteúdo
- Listas com mais de 10 itens
- Excesso de formatação

Descreva o cenário atual e os desafios.

- Ponto chave 1

- Ponto chave 3

### Implementação

3. Terceiro passo

> Citação relevante ou insight importante

## RESULTADOS

Mostre os resultados alcançados com dados concretos.

### Próximos Passos

- Ação futura 1

- Ação futura 3

Este é apenas o começo de nossa jornada sustentável.
