# Guia Simplificado: Como Criar um Post no Blog

## 📋 Exemplo Prático Usando o Texto "Carbono Negativo"

Este guia mostra como transformar um texto em um post completo no blog Gabardo, usando o formulário otimizado.

---

## 1. INFORMAÇÕES PRINCIPAIS ✅

### Título do Post *
```
Gabardo: A Primeira Transportadora Carbono Negativo do Mundo
```
✅ **Dica**: Use um título impactante que resuma a notícia principal

### URL Amigável (Slug) *
```
gabardo-primeira-transportadora-carbono-negativo
```
✅ **Gerado automaticamente** do título - pode editar se necessário

### Resumo Curto
```
A Transportes Gabardo fez história ao se tornar a primeira transportadora do mundo certificada como carbono negativo, removendo mais CO₂ da atmosfera do que emite.
```
✅ **Dica**: 1-2 frases que convidem à leitura (aparece nas listagens)

---

## 2. CLASSIFICAÇÃO E METADADOS 🏷️

### Categoria
```
Sustentabilidade ✓

📌 **Opções disponíveis**: Sustentabilidade, Inovação, Logística, Tecnologia, Transporte de Veículos, Segurança

### Tempo de Leitura
```
8 min

✅ **Ou especifique**: "João Silva", "Departamento de Sustentabilidade", etc.

## 3. IMAGEM DE CAPA 🖼️

### Recomendações
- **Tamanho ideal**: 1200x600 pixels
- **Formatos aceitos**: JPG, PNG, WebP
- **Peso máximo**: 10MB
- **Sugestão para este post**: Foto da frota Gabardo com elementos verdes/natureza

### Como enviar
1. Clique em "Selecionar Imagem"
2. Escolha o arquivo do seu computador
3. Aguarde o upload
4. Visualize o preview antes de salvar

## 4. CONTEÚDO DO POST ✍️

### Como formatar o texto

**OPÇÃO 1 - Texto Simples** (Recomendado para iniciantes)
Cole o texto completo e use quebras de linha para separar seções:

```
Gabardo: A Primeira Transportadora Carbono Negativo do Mundo

A Transportes Gabardo fez história ao se tornar a primeira transportadora do mundo certificada como carbono negativo. Esse marco representa não apenas um feito técnico, mas um compromisso profundo com o futuro da mobilidade sustentável e da logística verde.

O que significa ser uma transportadora carbono negativo

Ser carbono negativo vai além de neutralizar emissões. Significa que a empresa remove mais CO₂ da atmosfera do que emite. Ou seja, a Gabardo não apenas compensa suas operações, mas contribui ativamente para a regeneração do planeta.

Em 2024, após anos de inventários e ações estratégicas, a empresa atingiu um saldo negativo de –23.890,90 tCO₂e. Com um inventário total de 57.884,58 tCO₂e e um volume de créditos de carbono equivalente a 81.775,48 tCO₂e, a Gabardo consolidou uma operação logística que vai além do impacto neutro — tornando-se positiva para o planeta.

[Continue com o resto do texto...]
```

✅ **Dicas de Formatação**:
- Use TÍTULOS EM MAIÚSCULAS para seções principais
- Deixe uma linha em branco entre parágrafos
- Mantenha a estrutura natural do texto
- O sistema processa automaticamente

## 💡 DICAS EXTRAS

### Para Textos Longos
- Divida em seções claras com subtítulos
- Use parágrafos curtos (3-4 linhas)
- Destaque informações importantes
- Adicione números e dados quando relevante

### Para Imagens
- Sempre use imagens de alta qualidade
- Prefira fotos reais da Gabardo
- Evite imagens genéricas de banco de imagens
- Certifique-se que a imagem representa o conteúdo

### Para Tags
- Use no mínimo 4 tags
- Misture termos gerais e específicos
- Exemplo: "sustentabilidade" (geral) + "carbono negativo" (específico)
- Pense no que o leitor buscaria no Google

---

## 🆘 PROBLEMAS COMUNS

### "O slug não está sendo gerado"
- Digite o título primeiro
- O slug é gerado automaticamente
- Se quiser customizar, pode editar manualmente

### "A imagem não carregou"
- Verifique se o arquivo tem menos de 10MB
- Use formatos: JPG, PNG ou WebP
- Tente novamente (pode ser conexão lenta)

### "Erro ao salvar"
- Preencha todos os campos obrigatórios (*)
- Verifique sua conexão com internet
- Recarregue a página e tente novamente

---

## ✅ PRONTO PARA PUBLICAR!

Após preencher todos os campos, clique em:
- **"Publicar Post"** - para posts novos
- **"Salvar Alterações"** - para edições

O post estará disponível em: `https://seusite.com.br/blog/[seu-slug]`
